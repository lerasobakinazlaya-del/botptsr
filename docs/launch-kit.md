# Launch Kit

## Positioning

Нить - AI-компаньон в Telegram для живого диалога, поддержки, разбора и фокуса.

Мы не продаем "еще один чат-бот". Мы продаем ощущение нормального разговора: бот держит контекст, не повторяет один вопрос, подстраивается под темп и помогает продолжить мысль.

## Safe Public Offer

Использовать в рекламе:

- AI-компаньон в Telegram
- живой диалог без шаблонов
- память контекста
- приватный собеседник
- поддержка, разбор и фокус
- попробуй бесплатно, продолжай глубже в Pro/Premium

Не использовать в рекламе:

- adult/dating/sex positioning
- drugs, party, explicit fantasy copy
- обещания заменить отношения, терапию или врача
- кликбейт про личные качества пользователя

## 7-Day Launch Plan

Day 1:

- run `python scripts/prelaunch_check.py --launch-mode`
- run GitHub Actions workflow `Launch Readiness` against production
- run GitHub `Diagnose Bot`
- verify real payments or consciously keep traffic to free/test cohort
- prepare 10 links with start parameters: `src_tg__cmp_ai_live_dialogue`

Day 2:

- seed 5-10 small Telegram channels
- budget: 50-150 USD equivalent
- measure CTR, `/start`, first message, 3+ messages

Day 3:

- read first dialogues in admin
- fix first response and paywall text if second-message rate is weak
- publish 2 organic posts or short videos

Day 4:

- test 3 offers: "AI-компаньон", "приватный собеседник", "бот с памятью"
- stop placements with weak activation

Day 5:

- inspect paywall views and premium clicks
- adjust Pro/Premium wording
- launch retest with best 2 channels

Day 6:

- scale only the best source/campaign pair
- keep budget small until paid conversion is real

Day 7:

- report CAC, activation, paywall view, premium click, paid conversion, D1/D3 retention
- decide: scale, change offer, or fix product

## Metrics

Track:

- ad CTR
- start rate
- first message rate
- 3+ messages rate
- paywall view
- premium click
- paid conversion
- CAC
- D1/D3 retention
- second-message rate after first AI answer

The critical product metric is second-message rate. If users do not answer after the first bot reply, advertising will only buy churn.
