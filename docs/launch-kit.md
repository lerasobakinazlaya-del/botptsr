## BotFather

### Short Description

AI-партнёр с памятью диалога для поддержки, разбора и фокуса.

### Description Variant 1

Нить — бот для тех моментов, когда нужен не просто ответ от AI, а нормальный контакт.

Он помогает:

- выдохнуть и не развалиться
- разложить хаос по полкам
- быстро вернуть себе фокус

Внутри четыре сильных режима:

- Диалог — живой умный разговор без ассистентского лака
- Психолог — бережная опора для тревоги, перегруза и ПТСР-чувствительных состояний
- Разбор — ясность, структура и следующий шаг для сложных решений
- Фокус — короткий и твёрдый режим, когда нужно вернуть контроль

Premium открывает память диалога между сообщениями, инициативу от бота и все сильные режимы.

### Description Variant 2

Нить — личный AI-партнёр в Telegram.

Не просто отвечает на сообщения, а помогает держать нить разговора: помнит контекст, поддерживает в перегрузе, помогает принимать решения и возвращает в рабочее состояние.

Внутри четыре режима: Диалог, Психолог, Разбор и Фокус. Если нужен более глубокий формат работы, Premium открывает память диалога, инициативу от бота и все сильные режимы без ограничений preview.

### Description Variant 3

Нить — это AI-бот, к которому можно вернуться завтра и не начинать всё заново.

Он создан для людей, которым нужен не безличный чат-бот, а удобный инструмент для ежедневного контакта: выдохнуть, прояснить ситуацию, собрать решение, вернуть темп.

Внутри: Диалог для живого контакта, Психолог для бережной опоры, Разбор для ясности и Фокус для жёсткого темпа. Premium даёт память контекста, инициативу от бота после паузы и доступ ко всем режимам.
