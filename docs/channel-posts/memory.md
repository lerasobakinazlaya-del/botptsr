Большинство ботов каждый раз будто видят тебя впервые.

Нить строится вокруг другого ощущения: разговор не должен начинаться с нуля, если ты уже рассказывал важные детали. Память нужна не для "вау-эффекта", а чтобы бот не отвечал глупо на простые вопросы вроде "как меня зовут" и не терял нить диалога.

Память контекста особенно важна, когда разговор не помещается в один вечер: решение, тревога, проект, конфликт, длинная задача.
