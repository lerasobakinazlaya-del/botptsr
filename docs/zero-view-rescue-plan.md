# План выхода из нуля просмотров

Если у ролика 0 просмотров, сначала чиним не креатив, а доставку. Нулевые просмотры обычно означают одно из трех: аккаунт не прогрет, публикация ограничена настройками, или платформа режет ролик как внешний трафик/спам.

## Проверка перед публикацией

1. Аккаунт не должен быть приватным.
2. У видео должна быть публичная видимость: Public / Everyone.
3. Не публиковать 5-10 роликов подряд за 10 минут на новом аккаунте.
4. Не вставлять прямую ссылку `https://t.me/...` в описание первых роликов.
5. Не писать в каждом ролике одинаковую подпись и одинаковые хэштеги.
6. Не использовать чужой водяной знак в кадре.
7. Первые 2 секунды должны быть понятны без звука.

## Что делать первые 48 часов

День 1:

- Утром: оформить профиль, аватар, био, закрепить 1 ролик.
- Перед первым роликом: 10-15 минут живого поведения в приложении: посмотреть похожие видео, лайкнуть 5-7, оставить 2 нормальных комментария.
- Опубликовать 3 ролика с разницей 2-3 часа.
- После публикации не удалять ролик сразу, даже если 0 просмотров первые 30-60 минут.

День 2:

- Опубликовать еще 5-7 роликов.
- Каждый ролик менять по одному параметру: хук, музыка, первый кадр, подпись.
- Лучший ролик из Telegram/личного чата повторить в TikTok/Reels/Shorts разными подписями.

## Подписи без внешнего линка

Для TikTok:

```text
Ты тоже открываешь чат, когда не можешь уснуть?

#переписка #ночь #мысли #чат #ai
```

```text
Иногда нужно не решение. Нужно, чтобы кто-то помог собрать мысль.

#мысли #ночнойчат #переписка #нейросеть
```

```text
Не отправляй сразу. Сначала напиши как есть.

#сообщения #отношения #переписка #ночь
```

Для Reels:

```text
Когда мысль крутится по кругу, начни с одной фразы.

#переписка #мысли #вечер #ai
```

```text
Не всегда нужен совет. Иногда нужен спокойный ответ.

#ночь #переписка #мысли #telegram
```

Для Shorts:

```text
AI-собеседник для тех моментов, когда мысль не отпускает.

#ai #чатбот #переписка #shorts
```

## Когда добавлять Telegram

Первые 5-10 публикаций лучше делать без прямой ссылки в описании. Telegram добавляем мягко:

- в био профиля;
- в закрепленном комментарии;
- фразой `бот в профиле`;
- в конце ролика как `@asknitai_bot`, без длинной ссылки.

## Минимальный план публикаций

На один день:

- 3 ролика TikTok;
- 3 ролика Reels;
- 3 ролика Shorts;
- 1 пост в Telegram-канал;
- 2 сторис вручную.

Если аккаунт новый и все еще 0 просмотров после 24 часов, не заливаем больше подряд. Сначала проверяем настройки видимости, наличие ограничений аккаунта и пробуем один ролик через встроенный редактор платформы с нативной музыкой.
