# Шаблоны подписей

## TikTok / Reels

```text
{hook_short}. Проверь в Нити: ссылка в профиле.
```

```text
Обычный AI: {bad_pattern}. Нить: {better_pattern}. Попробуй бесплатно.
```

```text
Сохрани промпт: “{prompt}”. Отправь его боту по ссылке в профиле.
```

```text
Если задача большая, не режь её на куски. Нить держит контекст и продолжает мысль.
```

## Банк безопасных CTA

- Попробуй бесплатно.
- Проверь на своей задаче.
- Сохрани и отправь боту, когда будет длинный текст.
- Если нужен полный разбор, открой доступ на день.
- Ссылка в профиле.
