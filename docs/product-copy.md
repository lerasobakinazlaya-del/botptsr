# Product Copy

## Telegram

### Short Description

Личный AI-партнёр для поддержки, разбора и фокуса с памятью диалога.

### Full Description

Бот для тех моментов, когда нужен не просто ответ от AI, а нормальный контакт.

Он помогает в трёх главных сценариях:

- выдохнуть и не развалиться
- разложить хаос по полкам
- быстро вернуть себе фокус

Внутри:

- `Диалог` — живой умный разговор без ассистентского лака и коучинговой маски
- `Психолог` — бережная опора для тревоги, перегруза, ПТСР-триггеров и тяжёлых состояний
- `Разбор` — ясность, структура и следующий шаг для решений, работы, идей и сложных развилок
- `Фокус` — коротко, твёрдо и по делу, когда нужен темп, рамка и возвращение контроля

Premium открывает память диалога между сообщениями, инициативу от бота после паузы и все сильные режимы без ограничений preview.
