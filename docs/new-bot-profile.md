# Профиль нового Telegram-бота Нити

Этот файл — готовый набор текстов для BotFather и запуска нового бота в одном стиле.

## Название

Основной вариант:

```text
Нить
```

Если Telegram попросит более длинное имя:

```text
Нить · AI-собеседник
```

## Username

Проверять по очереди в BotFather:

```text
nit_ai_bot
nit_companion_bot
nit_dialog_bot
nit_memory_bot
nit_talk_bot
```

Лучший вариант для продвижения: `nit_ai_bot`, потому что коротко и понятно для ссылки.

## About

Короткое описание профиля:

```text
AI-собеседник, который держит контекст, помогает разобрать мысли и продолжает диалог без шаблонного тона.
```

## Description

Полное описание:

```text
Нить — живой AI-собеседник в Telegram.

Она помогает не потерять мысль: поддержать разговор, разобрать ситуацию, собрать фокус или продолжить длинную задачу без ощущения, что ты каждый раз начинаешь заново.

Можно писать как есть: тревога, хаос, рабочая задача, текст, идея или решение. Нить подхватит контекст и поможет довести разговор до ясности.
```

## Команды

Для BotFather `/setcommands`:

```text
start - начать диалог
modes - выбрать режим общения
premium - открыть платный доступ
help - как пользоваться Нитью
```

## Аватар

Использовать:

```text
assets/nit-avatar.png
```

Это главный образ Нити: темный премиальный портрет, световая нить, спокойный живой AI-компаньон.

## Стартовая карточка

Использовать в боте:

```text
assets/launch-card.png
```

Она уже подключена в `config/runtime_settings.json` как `ui.start_avatar_path`.

## Первый пост в канал

```text
Нить скоро в Telegram.

Это AI-собеседник, который не сбрасывает тебя в новый пустой чат, а держит контекст, настроение и ход мысли.

Можно прийти с тревогой, рабочей задачей, длинным текстом, идеей или просто фразой “я запутался”. Нить подхватит и поможет развернуть разговор.

Будь среди первых: https://t.me/<new_bot_username>
```

## Сообщение старому боту как мост

```text
Мы переносим Нить в новый бот, чтобы собрать продукт в одном стиле и дальше развивать его чище.

Новый вход здесь: https://t.me/<new_bot_username>

Там будет обновленный профиль, стартовая карточка, режимы и дальнейшие улучшения.
```

## Порядок действий

1. В BotFather создать нового бота и выбрать username.
2. Поставить `assets/nit-avatar.png` как аватар.
3. Заполнить About и Description текстами выше.
4. Добавить команды через `/setcommands`.
5. Обновить `BOT_TOKEN` в GitHub Secrets или на сервере.
6. Обновить `launch.bot_username` и ссылки в админке.
7. Запустить prelaunch и деплой.
8. Проверить `/start`, оплату, режимы и инициативные сообщения.
